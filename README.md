# 🍅 Simulador de Estufa Sustentável - Projeto Agrinho 2026

Repositório destinado ao projeto Agrinho - Um simulador interativo de cultivo de tomate em estufa com foco em sustentabilidade e controle ambiental.

## 📋 Sobre o Projeto

Este projeto foi desenvolvido para o **Programa Agrinho 2026** com o tema: *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*.

O simulador permite que os usuários compreendam como variáveis ambientais (temperatura, umidade do ar e do solo) afetam o crescimento de uma planta de tomate em uma estufa, promovendo educação sobre agricultura sustentável e tecnologia agrícola.

## 🎮 Funcionalidades

- **Simulador Interativo**: Controle temperatura, umidade do ar e solo em tempo real
- **Feedback Dinâmico**: Mensagens de alerta e recomendações sobre as condições
- **Barra de Progresso**: Visualize o crescimento da planta de forma clara
- **Múltiplas Páginas**: Navegação entre Simulador, Sobre Mim e Informações do Agrinho
- **Design Responsivo**: Interface adaptável para diferentes tamanhos de tela

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Ajuste os controles de temperatura e umidade
3. Clique em "Avançar 1 Dia" para simular o crescimento
4. Observe as mudanças na planta e nas mensagens de alerta
5. Encontre o equilíbrio ideal para um crescimento saudável!

## 🌿 Condições Ideais da Planta

| Variável | Mínimo | Ideal | Máximo |
|----------|--------|-------|--------|
| **Temperatura** | 10°C | 18-28°C | 40°C |
| **Umidade do Ar** | 20% | 60-80% | 100% |
| **Umidade do Solo** | 10% | 60-70% | 100% |
